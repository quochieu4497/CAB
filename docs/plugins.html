Fixed Header

Tính năng này có 2 class đặc biệt cho header sẽ được tự động gán là .active, .fixedheader

Class
.active
Class này sẽ tự động đổi màu (hoặc thuộc tính khác) của header khi scroll page và sẽ được bật nếu bạn chọn mục Active Header Tranparents và Header Position trong phần config site trên CAB
.fixedheader
Class này sẽ tự động gắn cứng header khi vào mobile hoặc scroll page và sẽ được bật nếu bạn chọn mục Active Fixed Header và Header Position trong phần config site trên CAB

Fixed Footer

Tính năng này sẽ tự động gắn cứng footer khi nội dung ngắn và sẽ được bật nếu bạn chọn mục Active Fixed Footer và Display footer trong phần config site trên CAB

Back to top

Tính năng này sẽ tự động hiển thị nút Back to top khi scroll page và sẽ được bật nếu bạn chọn mục Active Back to top button và Display ToTop trong phần config site trên CAB

Form auto validator

Tính năng này tự động đổi báo lỗi form khi người dùng nhập sai yêu cầu với việc sử dụng attribulte [data-toggle="validator"] trong thẻ form nó được sử dụng đi kèm với các attribulte [data-error] class .help-block, required:

Định dạng cho thẻ form
<form data-toggle="validator">
 <input data-error="Vui lòng nhập thông tin" required>
 <div class="help-block with-errors"></div>
 <input data-error="Vui lòng nhập thông tin" pattern="^[a-z0-9._%+-]+@[a-z0-9.-]+.[a-z]{2,3}$" required>
 <div class="help-block with-errors"></div>
</form>
Các pattern tham khảo
Email
^[a-z0-9._%+-]+@[a-z0-9.-]+.[a-z]{2,3}$
Phone
^[_A-z0-9]{1,}$
Password (UpperCase, LowerCase and Number)
^(?=.*\d)(?=.*[a-z])(?=.*[A-Z])(?!.*\s).*$
Password (UpperCase, LowerCase, Number/SpecialChar and min 8 Chars)
(?=^.{8,}$)((?=.*\d)|(?=.*\W+))(?![.\n])(?=.*[A-Z])(?=.*[a-z]).*$

Xem thêm tại http://html5pattern.com/

Image Zoom

Phóng to ảnh trong chi tiết sản phẩm

<article class="product-item">
 <img id="zoomimg" src="../img/product/01.jpg" data-zoom-image="../img/product/t-shirt.jpg">
</article>
<article class="product-control">
 <a class="item active" href="javascript:void(0)">
  <figure>
   <img src="../img/product/s-01.png" data-preview-image="../img/product/01.jpg" data-zoom-image="../img/product/t-shirt.jpg">
  </figure>
 </a>
</article>
$("#zoomimg").elevateZoom({
 responsive: true,
 gallery: "product-control",
 galleryActiveClass: "active"
});
$(".product-control .item").each(function () {
 $(this).click(function (e) {
  var currentValue = $(this).find('img').attr('data-preview-image')
  var currentValueZoom = $(this).find('img').attr('data-zoom-image')
  $('.product-control .item').removeClass('active')
  $(this).addClass('active')
  var gallery_zoomimg = $('#zoomimg').data('elevateZoom');
  gallery_zoomimg.swaptheimage(currentValue, currentValueZoom);
 });
});

Hover to change

data-change

data-old

data-new
URL ảnh hoặc tên class
