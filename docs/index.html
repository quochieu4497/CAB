Trước khi sử dụng bộ Kit này bạn phải hiểu rằng đây là tài sản riêng của Cánh Cam, bạn phải tuân thủ các qui định để sử dụng nó vào mục đích thích hợp. Công cụ có tên All-In-One Website Builder tool hay còn gọi là CAB

Dành cho Sales

Truy cập vào CAB sau đó nhập tên dự án cần tạo site

Ở màn hình chính bạn kéo thả components vào, bên cạnh đó bạn có thể tạo thêm nhiều page cho sites, Sau khi hoàn tất bạn nhấn nút Tạo site

Bạn có thể thay đổi màu sắc cũng như thuộc tính Javascript của site với bảng cấu hình bên dưới.

Dành cho Developers

Để cài đặt bộ kit này trước tiên bạn clone source về từ:

git clone https://github.com/baonguyenyam/canhcam-cab.git

Sau đó tiến hành cài đặt bằng cách nhấp vào file

install.bat

Nếu bạn sử dụng Mac thì có thể chạy file install.sh

Để đảm bảo công việc thuận lợi bạn phải xác định rõ mục tiêu sử dụng:
  • 1) Nếu bạn là lập trình viên: Máy bạn phải có sẵn NodeJS cũng như các plugins đi kèm như: pug, gulp, sass, npm...
  • 2) Nếu bạn là sales: Bạn chỉ có quyền tạo site và sau đó liên hệ phòng develop để copy source (nếu được duyệt)

Thực thi ứng dụng

Nếu sử dụng MacOS

Vào thư mục dự án gọi lệnh:

sh app.sh

Trong đó:

  • 1) Ấn số 1 + Enter để chạy công cụ CAB
  • 2) Ấn số 2 + Enter: Sau khi build xong web muốn xem thử chạy lệnh này
  • 3) Ấn số 3 + Enter: Để xây dựng thêm thư viện cho CAB, xem thêm tại đây
Nếu sử dụng Windows

Các sử dụng tương tự nhưng chạy file:

app.bat

Lệnh build trên CAB

1) Để chạy CAB bạn chạy lệnh

gulp builder && NODE_ENV=production node index.js

2) Sau khi build xong web muốn xem thử chạy lệnh

npm start

3) Để xây dựng thêm thư viện cho CAB bạn chạy lệnh

npm run dev

Lệnh build trên Project

Sau khi kéo thả và tạo site xong developer vào thư mục @SITE/[tên-dự-án] và tiến hành cài đặt với hai lệnh cơ bản:

- npm i
- bower update

Muốn run dev chạy lệnh

npm start

Chú ý: Có thể bạn phả chạy lại lệnh npm i sass node-sass để rebuild lại sass trong project

1) Để chạy Build tích hợp nén CSS,JS,HTML,IMG và deploy lên server bạn chạy lệnh

npm run pro

2) Để chạy Build KHÔNG NÉN và deploy lên server bạn chạy lệnh

npm run pro-no

3) Để chạy Build tích hợp nén CSS,JS,HTML,IMG và chạy được trên local bạn chạy lệnh

npm run local

4) Để chạy Build KHÔNG NÉN và chạy được trên local bạn chạy lệnh

npm run local-no
